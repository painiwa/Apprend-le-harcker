Leçon 2 : Commandes Linux de base

Introduction

Linux est le système d’exploitation le plus utilisé par les hackers éthiques, car il offre un contrôle complet du système et de nombreux outils de sécurité. Maîtriser les commandes de base est essentiel pour naviguer dans le système et exécuter des scripts ou tests de sécurité.

Commandes essentielles

  • ls : lister les fichiers et dossiers dans le répertoire courant.
  • cd : changer de répertoire pour naviguer dans l’arborescence.
  • pwd : afficher le chemin complet du répertoire courant.
  • mkdir : créer un nouveau dossier.
  • rm : supprimer des fichiers ou dossiers.
  • touch : créer un fichier vide.

Exemples pratiques


# Lister tous les fichiers, y compris cachés, avec détails
ls -la

# Changer de répertoire pour aller dans Documents
cd Documents

# Afficher le chemin complet du répertoire courant
pwd

# Créer un nouveau dossier appelé "Tests"
mkdir Tests

# Créer un fichier vide "exemple.txt"
touch exemple.txt

# Supprimer un fichier
rm exemple.txt

# Supprimer un dossier vide
rmdir Tests

Explications ligne par ligne

  • ls -la : -l affiche les détails (permissions, propriétaire, taille), -a montre tous les fichiers y compris cachés.
  • cd Documents : change le répertoire vers "Documents". Vous pouvez utiliser cd .. pour revenir en arrière.
  • pwd : affiche le chemin complet, utile pour savoir où vous êtes dans l’arborescence.
  • mkdir Tests : crée un dossier appelé "Tests".
  • touch exemple.txt : crée un fichier vide pour tester des scripts ou commandes.
  • rm exemple.txt : supprime le fichier. Attention : cette action est définitive.

Conseils pour débutants

  • Apprenez d’abord à naviguer dans l’arborescence avec cd, pwd et ls.
  • Faites des tests sur un dossier ou fichier factice avant d’utiliser rm sur vos fichiers importants.
  • Combinez les commandes avec des options pour mieux explorer le système.
  • Entraînez-vous régulièrement dans un terminal Linux ou via une machine virtuelle.

Résumé

Cette leçon vous a présenté les commandes Linux de base essentielles pour un hacker éthique. Ces commandes permettent de :

  • Naviguer efficacement dans le système.
  • Créer et gérer des fichiers et dossiers.
  • Préparer des tests de sécurité dans un environnement contrôlé.
