Leçon 4 : Sécurité Web

Introduction

La sécurité web est cruciale pour protéger les applications contre les attaques courantes. Les hackers éthiques doivent comprendre les vulnérabilités pour pouvoir les détecter et les corriger.

Principales vulnérabilités web

  • Injection SQL : Permet à un attaquant de manipuler la base de données via des requêtes non sécurisées.
  • Cross-Site Scripting (XSS) : Injection de scripts malveillants dans une page web visitée par d’autres utilisateurs.
  • Cross-Site Request Forgery (CSRF) : Forcer un utilisateur authentifié à exécuter des actions non désirées sur un site web.

Exemple pratique : Échappement de caractères en PHP


// Exemple simple pour sécuriser l'entrée utilisateur en PHP
// htmlspecialchars() convertit les caractères spéciaux en entités HTML
$user_input = htmlspecialchars($_POST['input'], ENT_QUOTES, 'UTF-8');

// Affichage sécurisé
echo "Vous avez saisi : " . $user_input;

Explications ligne par ligne

  • $_POST['input'] : récupère la valeur envoyée par un formulaire.
  • htmlspecialchars() : transforme les caractères spéciaux (ex : <, >, &) pour empêcher l’injection de code HTML ou JS.
  • Le deuxième paramètre ENT_QUOTES protège également les guillemets simples et doubles.
  • Le troisième paramètre 'UTF-8' assure un encodage correct pour tous les caractères.

Conseils pour débutants

  • Ne jamais afficher directement les données utilisateur sans les filtrer ou échapper.
  • Utiliser des requêtes préparées pour éviter les injections SQL.
  • Tester vos applications avec des outils comme OWASP ZAP ou Burp Suite en environnement sécurisé.
  • Documenter toutes les vulnérabilités et solutions dans vos rapports de test éthique.

Résumé

Cette leçon vous a présenté les vulnérabilités web les plus courantes et comment les prévenir. Vous devez maintenant être capable de :

  • Identifier les injections SQL et XSS dans un code source.
  • Échapper correctement les entrées utilisateur en PHP.
  • Appliquer des pratiques de sécurité pour protéger vos applications web.
